# 🎭 Meme Guesser Hub
> Автоматический угадыватель мемов для Roblox

---

## ⚡ Быстрый запуск

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/AretoGoto/MemOrLavaHub/refs/heads/main/main.lua"))()
```

---

## 🎮 Функции

### 🎵 Угадывание мемов
- Автоматически определяет правильный ответ
- Физически нажимает на кнопку ответа
- Рандомная задержка для скрытности

### 🎁 Авто награды
- Каждые 60 секунд проверяет доступные награды
- Автоматически забирает все доступные награды за игровое время

### 📦 Быстрые кейсы
- Убирает анимацию прокрутки при открытии кейса
- Моментальный результат

---

## ⚙️ Настройки

| Настройка | Описание | По умолчанию |
|-----------|----------|--------------|
| Авто угадывание | Включить/выключить автоответ | Выкл |
| Мин задержка | Минимальное время перед ответом | 0.3 сек |
| Макс задержка | Максимальное время перед ответом | 1.3 сек |
| Авто награды | Автосбор наград за игровое время | Выкл |
| Быстрые кейсы | Убрать анимацию открытия | Выкл |
